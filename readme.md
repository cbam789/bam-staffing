# Bam Staffing App

**Bam Staffing** is a custom-built scheduling tool that compares hourly demand to available staffing capacity and provides optimized staffing recommendations.

## 📁 Project Structure


## 📊 Features

- Hourly demand vs. coverage comparison
- Staff recommendation engine based on expected productivity
- PDF export with company branding and cover page
- Interactive GUI for file input (coming soon)
- EXE build for standalone use (planned)

## 📎 Reports

Generated reports are stored in the `reports/` folder. These include:

- `bam_staffing_summary.pdf`: Includes a visual chart and recommended staffing output.
- `cover_letter.txt`: Brief summary of the report’s purpose.

## 🚀 Getting Started

> Make sure Python is installed and run the app from `src/main.py`. More features coming soon!
